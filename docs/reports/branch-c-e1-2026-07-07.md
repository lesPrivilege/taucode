# Branch C E1 Report (2026-07-07)

## Scope

Branch C tests whether a fixed short projected-turn nudge can reproduce the `C+PL` completion signal without semantic anchor or sideband machinery.

Real DeepSeek runs, `G2-E1`, `compact-after=32000`, `keep-recent=3`, n=3 per arm:

- `C''`: C + semantic anchor (`ECODE_SEMANTIC_ANCHOR=1`)
- `C+N`: C + fixed compact nudge tail

Artifacts:

- JSONL: `experiments/results/branch-c-e1/*.jsonl`
- Compare: `experiments/results/branch-c-e1/compare-branch-c.json`

## Implementation Check

`C+N` uses a fixed byte-stable tail:

```text
[nudge]
Goal; verify files; finish.
```

The nudge is projected-turn only, default-off, and recorded through the same OBS-TAIL mechanism as anchors/placebo. Mock preflight produced 6/6 projected turns with `source:"nudge"` and hash `21ec07ca47f7b489`.

## R15 Matrix Read

This run falls in the direction: `C+N` improves completion coverage but increases exploration/cost sharply.

Do not read this as significance. The directional result is:

- The short projected-turn cue may help completion reliability on this packet.
- The current cue also appears to encourage prolonged work loops: two runs went to 84/86 turns and >3M total tokens.
- The benefit is not a cheap replacement for the anchor as-is.

## Cost Shape

| Arm | Median input | Median output | Median reasoning | Median re-reads | Median compacted-path re-reads |
| --- | ---: | ---: | ---: | ---: | ---: |
| `C''` | 528,591 | 10,672 | 1,761 | 6 | 3 |
| `C+N` | 3,205,160 | 24,896 | 5,168 | 9 | 9 |

The nudge tail itself is tiny. The cost comes from changed behavior: more turns, more projections, and more repeated context interactions.

## Verdict

`C+N` is directionally positive for completion in this batch, but not acceptable as a production branch without a budget governor. It supports the hypothesis that a projected-turn orientation cue can matter, while also showing that the cue must be paired with stop/budget discipline.

Recommended next branch:

- Keep README frozen.
- Do not promote raw `C+N`.
- Test a bounded variant: fixed nudge plus max projected-turn count, max reread budget, or a "finish now if acceptance target is satisfied" termination cue.
- Keep semantic sideband paused until summary dedupe and output caps are implemented.
