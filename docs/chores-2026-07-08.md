## CHORE-1 · 清除 docs/scratch/（过时预览生成器）

**背景**：`docs/scratch/compile.js` 生成的是第一代互动双语预览页
（commit 8eedc19 一系），已被 arxiv 静态版取代（7a39dbb）。现状不可运行：
`package.json` 声明 `"type": "commonjs"`，`compile.js` 却用 ESM import
（marked v18 亦 ESM-only）。目录含整套 node_modules（492K），未追踪。

**任务**：`rm -rf docs/scratch/`。
**验收**：目录不存在；`git status --short` 无 docs/ 相关输出。
**禁区**：不碰 docs/ 其他文件。若目录内存在 compile.js / package.json /
package-lock.json / node_modules 之外的文件，停手报告，不删。

## CHORE-2 · 论文预览页确定性编译脚本（消除手工重生成）

**背景**：现行 `docs/index.html`（arxiv 静态风格，~230K，内容内嵌）没有
生成器——repo 内唯一构建脚本是 CHORE-1 要删的过时版。论文 md 每次修订
都要手工/agent 重排 HTML。对一个命题是「编译优于解释执行」的 repo，
论文预览自己走解释执行路线是自相矛盾，此件兼有卫生与自洽双重性质。

**输入**：`docs/paper-context-economy-2026-07-07.md`（EN）、
`docs/paper-context-economy-2026-07-07_zh.md`（ZH）、现行 `docs/index.html`
（版式模板来源：逐字提取其 CSS 与页面骨架，不重设计）。

**任务**：
1. 新建 `docs/tools/preview/`：`build.mjs` + `package.json`（pin marked；
   node_modules 已被根 .gitignore 的 `**/node_modules/` 覆盖）。
2. `build.mjs`：读两份 md → 按标题层级分节对齐（EN/ZH 节数不齐必须
   非零退出并指出错位处，不静默补空节）→ 套现行 index.html 版式生成
   同款三模式页（EN / 中文 / 对照）→ 写 `docs/index.html`。
3. 确定性：同输入两次运行，输出逐字节相同（禁时间戳、随机 id）。
4. 自包含：输出不得引入任何 CDN / 外部资源（现行页已无，保持）。
5. 跑一次，再生成的 index.html 与脚本一并提交——脚本自此为唯一生成路径。

**验收**：连跑两次 `node docs/tools/preview/build.mjs`，第二次
`git diff docs/index.html` 为空；产出页三模式人眼可用、TOC 可跳转；
页内无 `<link href="http` / `<script src="http` 类外部请求；
人为在 ZH md 删一个标题后运行，非零退出。
**禁区**：不改两份论文 md 的内容；不重设计版式；不引入 md 之外的内容源；
不改 CHORE-1 范围（scratch 由其自行处理，两件无依赖）。

**任务**：

2. 迁出后从 pi/ 删除三个未追踪项。

新 docs 文件含原三份全部非重复内容；taucode 根 `git status` 仅新增该文件。
